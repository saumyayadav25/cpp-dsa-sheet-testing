Test POTD Sync Functionality

This is a simple test to verify that the POTD sync works correctly.

Test Instructions:

  1. Go to the DSA sheet page (http://localhost:3000/sheet)
  2. Note the current POTD question and check if it's marked as solved in the DSA list
  3. Click "Mark as Done" on the POTD
  4. Verify that the corresponding question in the DSA list is now checked
  5. Refresh the page and verify the sync persists
  6. Verify that checking a DSA question directly does NOT affect the POTD status

Expected Behavior:

  • ✅ Marking POTD as done should check the corresponding DSA list question
  • ✅ If the same question appears again as POTD on another day, it should NOT be auto-marked
  • ✅ Checking a DSA list question directly should NOT affect POTD status
  • ✅ Changes should persist after page refresh

Test Data Storage:

Open browser DevTools -> Application -> Local Storage to see:

  • potd_last_done - stores when POTD was last completed
  • potd_streak - stores current streak
  • dsa-progress - stores DSA question progress with keys like "3-7" (topicId-questionId)
